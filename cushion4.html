Concept
役割を脱ぎ捨て、自分に戻る場所

現場で神経をすり減らす看護師たちが、自分の本当の呼吸を取り戻せるような動線設計と視覚表現を目指しました。情報の透明性と視覚的な静寂を両立させ、一人の人間として未来を選び直すための場所を構築しています。

誰を、認めるか

夜勤明けの静かな朝。誰にも言えない疲れを抱えながら、それでも誰かのために尽くしてきた。そんな一人の看護師が、ふと立ち止まれる場所を目指しました。

設計の、眼差し

情報のノイズを削ぎ落とし、視覚的な静寂を。看護師としてバイタルを測る時の集中力に似た、相手のわずかな変化を肯定するための余白を設計に込めました。

Conclusion

「誰かをケアする人が、自分に戻れる場所。」

来訪者の痛みと、制作者の祈り。その両端を繋ぐことで、
このサイトは一人の「生存」を支える静かな居場所として機能します。

Technical Identity

01

Design / Figma

「受容の場所」を視覚化するために、余白設計と色彩心理を徹底しています。Figmaでのコンポーネント管理により、一貫性と拡張性のあるデザインを構築します。

02

Coding / GitHub

静かな読書体験を妨げない、滑らかなインタラクションを実装。GitHubを用いたバージョン管理と、セマンティックなマークアップで「誠実な構造」を目指します。

03

Direction

看護師として培った「傾聴力」を活かし、作り手の想いを解読。ただ作るのではなく、届けたい相手に届くための動線と、情報の優先順位を整理します。
