Overview

里山キャンプサイト

Role
Design / Coding / Planning
Tools
Figma / VS Code
Concept
伝説の夏休み。

都会で育つ子供たちへ、一生モノの「本物の手触り」を届ける里山体験プログラムの公式サイトです。
画面の中の疑似体験ではなく、五感をフルに使い、時には泥にまみれる泥臭い体験が、子供たちの将来を支える強い「根っこ」になることを願い、設計しました。

わが子に「本物の体験」をさせたい保護者様

「ゲームやYouTubeばかりの日常から、一度子供を連れ出したい。」
「自然の中で、たくましく成長するわが子の姿を見たい。」
「怪我や安全面は大丈夫か？」

設計の、眼差し子供たちの「生きる力」を育みたい主催者様

「『ねっこの学校』は、ただのレジャーではなく、一生忘れない『伝説の夏休み』を提供したい。 このサイトを通じて、五感で感じる自然の尊さと、専門スタッフが見守るからこそできる『挑戦』の価値を届けてほしいと依頼しました。」

Conclusion

「画面の中にはない、本物の手触り。」

ポップで元気なビジュアルを採用しつつ、
「信頼と安全」を感じさせる情報を整理。
楽しさと安心感が共存するデザインで子供にとってはワクワクする冒険の入り口、
親にとっては安心して子供を預けられる信頼の窓口として機能させました。
